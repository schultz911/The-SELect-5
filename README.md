# Top-Resolution-SEL
**Custom SEL to use with TamTaro**

Simple enough to understand. It will always display the top five streams, and _**only**_ the top five streams, based on their resolution after all of TamTaro's and Vidhin's handiwork.

> [!IMPORTANT]
>
> In order for this to work as intended, you must edit the sorting order as follows:
> - Seadex
> - Resolution
> - Quality
> - Visual Tag
> - Audio Tag
> - Stream Expressions
> - Stream Expression Score
> - Everything else...
